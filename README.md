# Dashlytics – Social Media Analytics Dashboard

## Live Website
https://kk-git-15.github.io/24BCE2788_AnalytiX/

## Project Overview

Dashlytics is a frontend social media analytics dashboard designed to visualize engagement statistics, follower growth trends, content performance, and scheduling insights. The dashboard provides an intuitive interface for analyzing social media data through interactive charts and structured analytics panels.

This project was developed using HTML, CSS, and JavaScript with Chart.js for data visualization.

---

## Features

### Dashboard Overview
- Engagement statistics display
- Follower growth tracking
- Monthly performance insights
- KPI metric cards

### Analytics Section
- Engagement distribution charts
- Platform performance comparison
- Monthly engagement trends
- Average likes and comments calculation

### Content Management
- Top performing posts table
- Engagement ranking system
- Scheduled posts display
- Performance classification (High/Medium/Low)

### UI Features
- Dark / Light theme toggle
- Sidebar navigation system
- Smooth section transitions
- Modern dashboard layout
- Interactive charts
